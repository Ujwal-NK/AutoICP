FAQ

Are my credentials safe Credentials are encrypted and stored locally on your device. The developers or third-party does not have access to your credentials.
It is encrypted using a randomly generated key, also stored locally on your device.
The password is stored in the .shadow file and the encryption key is stored in the .key file
I want to change my credentials Version 0.0.2
Open the root folder, delete the .shadow file
Restart the app, enter your username and password

Version 0.0.3
Open the settings menu
Enter the Username and password
Click on Save
I accidentally closed the app, did it log out Version 0.0.2
This version of the app does not support the Auto LogOut Feature.

Version 0.0.3
Auto LogOut feature ensures that you LogOut even in case you accidentally close the app.
NOTE: Does NOT LogOut if the app was force closed from the task manager
Can I Debug the app? The .log file contains all the necessary debug information. The log file does NOT contain any sensitive information.
This file can be shared with anyone for debugging or (must)while filing a error on GitHub
App says incorrect password, but it is working fine on the portal Delete the .key file and relaunch the app. Enter your credentials, without any spaces, and recheck.
The app unexpectedly closes Unhandled errors can cause the app to unexpectedly close.
This can also be caused if the app does not have the required permission, try running the app in administrator mode.
How do I log out on the version 0.0.2 On the terminal, Press Enter to Exit is displayed
Hit Enter to logout of the internet.
Version 0.0.2, app does not close on hitting Enter key A bug, that is fixed in the Version 0.0.3, you can close the app manually once you hit Enter.
How to download Selenium Driver? Open the driver page, to get all the necessary details regarding the selenium driver
Download the required driver to the app folder and extract it.
App suddenly stopped working The browser may have auto updated. Download the Selenium driver for the required version using this page and replace the driver.
Getting Started (Developers)
  • Download the source code and extract it into a folder.

  • Create a virtual environment in the same folder using python3
    python3 venv your_folder_name

  • Activate the environment
    . ./bin/activate #Linux
    ./Scripts/activate #Windows

  • Install the requirements from the requirements.txt file
    python3 -m pip install -r requirements.txt

  • Execution:
    python3 ./src/main.py

Getting Started

Version 0.0.2

  • Move the file into a folder
  • Download the selenium driver for the browser of your choice to the same folder and extract
  • Run the app, enter the credentials, if asked (only the first time)
  • Connected to the Internet
  • Hit Enter to logout of the Walled Garden
  • Close the app manually
Thereafter, launch the app to automatically connect to the internet

Version 0.0.3
  • Extract the zip file
  • Download the selenium driver for the browser of your choice, to the same folder and extract
  • Run the AutoINet file
  • Enter the credentials in the popup settings window
  • Click on save
  • Click on Disconnect to logout of the Walled Garden
Thereafter, launch the app to automatically connect to the internet
